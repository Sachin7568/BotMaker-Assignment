# BotLeague - Robotics Competition Platform

A modern, high-performance frontend landing page designed for the BotLeague national robotics competition. This project features a stunning dark-mode aesthetic with vibrant red accents, smooth scroll animations, and interactive component layouts.

## 🚀 Features
- **Dynamic User Journey**: Visual timeline showing the path from building a team to national rankings.
- **Competition Disciplines**: Interactive cards showcasing various robotics sports (Robo War, FPV Drones, RC Racing, etc.) with grayscale-to-color hover effects.
- **Live Event Brackets**: Integrated tournament bracket visualizations.
- **Ecosystem Registration**: Streamlined forms for judges, volunteers, and community members.
- **Fully Responsive**: Custom-built with Tailwind CSS to look great on desktop, tablet, and mobile devices.

## 🛠️ Technology Stack
- **React 18**
- **Vite** (for fast development and builds)
- **Tailwind CSS** (for styling and animations)

## 📦 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Sachin7568/BotMaker-Assignment.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Build for production:
   ```bash
   npm run build
   ```

## 🎨 Design System
- **Colors**: Deep dark backgrounds (`#000000` & `#141414`), accented with vibrant red (`#FF4C4C`) and custom blue borders.
- **Typography**: Uses 'Orbitron' for bold, modern headers and 'Roboto' for clean, readable body text.
